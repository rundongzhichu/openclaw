## 一、插件系统概述

### 1.1 设计目标

OpenClaw 插件系统旨在实现以下目标:

1. **可扩展性**: 支持第三方开发者轻松添加新的消息渠道和 AI 模型提供者
2. **隔离性**: 插件与核心系统通过明确定义的边界交互，避免紧耦合
3. **类型安全**: 完整的 TypeScript 类型定义，提供编译时检查
4. **向后兼容**: 版本化的契约设计，保证现有插件不被破坏
5. **易于调试**: 完善的日志和诊断工具

### 1.2 插件分类

OpenClaw 支持以下几种主要类型的插件:

#### 1.2.1 Channel 插件 (消息渠道适配器)

**作用**: 将不同消息平台 (WhatsApp, Telegram, Slack 等) 的消息统一适配到 OpenClaw 内部协议

**示例**:
- `extensions/whatsapp` - WhatsApp 渠道
- `extensions/telegram` - Telegram 渠道
- `extensions/slack` - Slack 渠道
- `extensions/discord` - Discord 渠道

**核心能力**:
```typescript
type ChannelPlugin = {
  id: ChannelId;              // 唯一标识
  meta: ChannelMeta;          // 元数据
  capabilities: ChannelCapabilities;  // 能力声明
  configSchema: ChannelConfigSchema;  // 配置 Schema
  setupWizard?: ChannelSetupWizard;   // 设置向导 (可选)
  
  // 各种功能适配器
  gateway: ChannelGatewayAdapter;     // 网关适配器
  outbound: ChannelOutboundAdapter;   // 发消息适配器
  pairing: ChannelPairingAdapter;     // 配对适配器
  directory: ChannelDirectoryAdapter; // 目录适配器
  // ... 更多适配器
};
```

#### 1.2.2 Provider 插件 (AI 模型提供者)

**作用**: 接入不同的 AI 模型 API (OpenAI, Anthropic, Ollama 等)

**示例**:
- `extensions/openai` - OpenAI/Codex
- `extensions/anthropic` - Anthropic Claude
- `extensions/ollama` - Ollama 本地模型
- `extensions/zai` - Z.ai (智谱 AI)

**核心能力**:
```typescript
type ProviderPlugin = {
  id: string;                 // 提供者 ID
  label: string;              // 显示名称
  docsPath: string;           // 文档路径
  aliases?: string[];         // 别名
  envVars?: string[];         // 环境变量
  
  auth: ProviderAuthMethod[]; // 认证方法
  catalog: ProviderCatalog;   // 模型目录
  
  // 可选的扩展能力
  streaming?: StreamingSupport;
  tools?: ToolSupport;
  webSearch?: WebSearchSupport;
};
```

#### 1.2.3 CLI Backend 插件

**作用**: 提供 CLI 命令的后端执行能力

**示例**:
- `system.run` - 系统命令执行
- `file.read` / `file.write` - 文件操作
- `browser.use` - 浏览器自动化

#### 1.2.4 Hook 插件

**作用**: 在 Agent 生命周期中注入自定义逻辑

**可用 Hooks**:
```typescript
const AVAILABLE_HOOKS = [
  "before_model_resolve",    // 模型解析前
  "before_prompt_build",     // 提示词构建前
  "before_agent_start",      // Agent 启动前
  "subagent_spawning",       // Subagent 孵化前
  "subagent_spawned",        // Subagent 孵化后
  "subagent_ended",          // Subagent 结束后
  "channel_message_received",// 收到渠道消息
  "gateway_started",         // Gateway 启动后
  "gateway_stopping",        // Gateway 停止前
] as const;
```

## 二、插件架构设计

### 2.1 分层架构

```
┌─────────────────────────────────────────────────────────┐
│                  Extensions (插件层)                     │
│  - extensions/whatsapp, telegram, slack, discord...     │
│  - extensions/openai, anthropic, ollama, zai...         │
│  - 每个插件有独立的 package.json 和源码目录               │
└─────────────────────────────────────────────────────────┘
                          ↓ 通过 Plugin SDK 交互
┌─────────────────────────────────────────────────────────┐
│              Plugin SDK (插件开发工具包)                 │
│  - src/plugin-sdk/* (公共类型和工具)                     │
│  - 子路径导出：plugin-sdk/core, plugin-sdk/runtime 等    │
│  - 禁止插件直接 import src/** (除 plugin-sdk 外)         │
└─────────────────────────────────────────────────────────┘
                          ↓ 注册和调用
┌─────────────────────────────────────────────────────────┐
│              Core System (核心系统)                      │
│  - src/plugins/* (插件发现、加载、注册表)                │
│  - src/channels/* (Channel 核心实现，非插件代码)         │
│  - src/gateway/* (网关控制平面)                          │
└─────────────────────────────────────────────────────────┘
```

### 2.2 边界规则

#### 2.2.1 插件 → 核心 (允许)

```typescript
// ✅ 允许：通过 Plugin SDK 导入
import type { ChannelPlugin } from "openclaw/plugin-sdk";
import { createChatChannelPlugin } from "openclaw/plugin-sdk/core";
import { dispatchReply } from "openclaw/plugin-sdk/reply-runtime";

// ✅ 允许：通过专用子路径导入
import { retryWithBackoff } from "openclaw/plugin-sdk/retry-runtime";
import { chunkText } from "openclaw/plugin-sdk/text-runtime";
```

#### 2.2.2 插件 → 核心 (禁止)

```typescript
// ❌ 禁止：直接导入核心内部模块
import { something } from "src/channels/xxx";
import { gatewayInternal } from "src/gateway/server.impl";

// ❌ 禁止：深导入 bundled 插件内部
import { helper } from "extensions/slack/src/utils";
import onboard from "extensions/acp/onboard.js";
```

#### 2.2.3 核心 → 插件 (限制)

```typescript
// ✅ 允许：通过暴露的 API 访问
const pluginApi = await loadPlugin("slack");
const result = await pluginApi.publicMethod();

// ❌ 禁止：深导入插件内部
import internal from "extensions/slack/src/internal-module";
```

### 2.3 插件注册流程

```typescript
// 1. 插件入口文件 (extensions/whatsapp/index.ts)
import { definePluginEntry } from "openclaw/plugin-sdk/plugin-entry";

export default definePluginEntry({
  id: "whatsapp",
  name: "WhatsApp Messenger",
  description: "WhatsApp 渠道适配器",
  
  register(api: OpenClawPluginApi) {
    // 2. 注册 Channel
    api.registerChannel({
      plugin: whatsappPlugin,
    });
    
    // 3. 注册 Provider (如果有)
    // api.registerProvider({ ... });
    
    // 4. 注册 Hooks
    // api.registerHook("before_agent_start", handler);
  },
});

// 5. 核心系统加载
// src/plugins/loader.ts
async function loadPlugin(pluginPath: string) {
  const module = await import(pluginPath);
  const pluginDef = module.default;
  
  // 调用 register 方法
  pluginDef.register(pluginApi);
}
```

## 四、Channel 插件开发

### 4.1 Channel 插件完整结构

```typescript
/**
 * @fileoverview WhatsApp Channel 插件
 * 
 * 功能:
 * 1. 监听 WhatsApp 消息
 * 2. 转发到 Gateway
 * 3. 接收 Gateway 指令并回复
 * 4. 支持群组、私聊、线程绑定
 */

import type { ChannelPlugin } from "openclaw/plugin-sdk";
import { createChatChannelPlugin } from "openclaw/plugin-sdk/core";
import { WhatsAppClient } from "./client.js";
import { processWhatsAppMessage } from "./message-processor.js";
import { sendWhatsAppReply } from "./reply-sender.js";

// ==================== 插件定义 ====================

const whatsappPlugin: ChannelPlugin = {
  // 基础信息
  id: "whatsapp",
  meta: {
    name: "WhatsApp Messenger",
    version: "1.0.0",
    author: "OpenClaw Team",
  },
  
  // 能力声明
  capabilities: {
    dm: "pairing",      // 私聊需要配对
    group: "open",      // 群组开放
    threading: true,    // 支持线程
    reactions: false,   // 不支持表情回应
  },
  
  // 配置 Schema
  configSchema: {
    schema: {
      type: "object",
      required: ["phoneNumber", "apiKey"],
      properties: {
        phoneNumber: {
          type: "string",
          pattern: "^\\+\\d+$",
          title: "WhatsApp 电话号码",
        },
        apiKey: {
          type: "string",
          title: "Cloud API Key",
        },
      },
    },
    uiHints: {
      phoneNumber: {
        label: "WhatsApp 号码",
        help: "格式：+8613800000000",
        placeholder: "+86...",
      },
      apiKey: {
        label: "API Key",
        sensitive: true,
        help: "从 Meta Developer Dashboard 获取",
      },
    },
  },
  
  // 设置向导 (可选)
  setupWizard: {
    choiceId: "whatsapp-cloud-api",
    choiceLabel: "WhatsApp Cloud API",
    steps: [
      {
        type: "input",
        field: "phoneNumber",
        label: "WhatsApp 号码",
        validation: (value) => /^+\d+$/.test(value),
      },
      {
        type: "input",
        field: "apiKey",
        label: "API Key",
        sensitive: true,
      },
    ],
  },
  
  // 功能适配器实现
  gateway: {
    startAccount: async (ctx) => {
      // 启动账号监听
      const client = new WhatsAppClient(ctx.account.config);
      await client.connect();
      
      client.on("message", async (msg) => {
        // 处理收到的消息
        const envelope = await processWhatsAppMessage(msg, ctx);
        await ctx.gateway.broadcast(envelope);
      });
      
      return () => client.disconnect();
    },
  },
  
  outbound: {
    send: async (ctx, payload) => {
      // 发送消息
      await sendWhatsAppReply(ctx.target, payload, ctx.account.config);
      return { success: true };
    },
  },
  
  pairing: {
    generatePairingCode: async () => {
      // 生成 6 位配对码
      return Math.floor(100000 + Math.random() * 900000).toString();
    },
    
    approvePairing: async (ctx, code) => {
      // 批准配对
      addToAllowlist(ctx.senderId, code);
    },
  },
  
  directory: {
    listPeers: async (ctx) => {
      // 列出联系人
      return await ctx.account.client.getContacts();
    },
    
    listGroups: async (ctx) => {
      // 列出群组
      return await ctx.account.client.getGroups();
    },
  },
};

export default whatsappPlugin;
```

### 4.2 Channel 适配器详解

Channel 插件通过**适配器模式**实现模块化，每个适配器负责一个功能领域:

#### 4.2.1 Gateway Adapter (网关适配器)

```typescript
type ChannelGatewayAdapter<ResolvedAccount = any> = {
  /**
   * 启动账号监听
   * 
   * @param ctx - 启动上下文
   * @returns 清理函数 (可选)
   */
  startAccount: (ctx: {
    cfg: OpenClawConfig;
    accountId: string;
    account: ResolvedAccount;
    runtime: RuntimeEnv;
    abortSignal: AbortSignal;
    log?: ChannelLogSink;
    getStatus: () => ChannelAccountSnapshot;
    setStatus: (next: ChannelAccountSnapshot) => void;
  }) => Promise<void | (() => void)>;
};
```

**职责**:
- 建立与消息平台的连接
- 监听 incoming messages
- 转换为 OpenClaw 内部事件格式
- 广播到 Gateway

#### 4.2.2 Outbound Adapter (发消息适配器)

```typescript
type ChannelOutboundAdapter = {
  /**
   * 发送消息到渠道
   * 
   * @param ctx - 发送上下文
   * @param payload - 回复负载
   * @returns 发送结果
   */
  send: (
    ctx: {
      target: MessagingTarget;
      account: ResolvedAccount;
      config: OpenClawConfig;
    },
    payload: ReplyPayload
  ) => Promise<SendResult>;
};
```

**职责**:
- 接收 Gateway 的回复指令
- 转换为渠道特定格式
- 调用渠道 API 发送
- 返回发送结果

```typescript
type ChannelPairingAdapter = {
```

### 4.3 使用 Helper 创建 Channel 插件

为简化开发，Plugin SDK 提供了 `createChatChannelPlugin` 辅助函数:

```typescript
import { createChatChannelPlugin } from "openclaw/plugin-sdk/core";
import type { ChannelPlugin } from "openclaw/plugin-sdk";

// 定义插件基础信息
const basePlugin = {
  id: "telegram",
  meta: { name: "Telegram", version: "1.0.0" },
  capabilities: { dm: "pairing", group: "open", threading: true },
};

// 使用 Helper 创建完整插件
const telegramPlugin: ChannelPlugin = createChatChannelPlugin({
  ...basePlugin,
  
  // 配置 Schema
  configSchema: buildChannelConfigSchema({
    required: ["botToken"],
    properties: {
      botToken: {
        type: "string",
        title: "Bot Token",
        description: "从 @BotFather 获取",
      },
    },
    uiHints: {
      botToken: {
        label: "Bot Token",
        sensitive: true,
        help: "联系 @BotFather 创建机器人",
      },
    },
  }),
  
  // Gateway 适配器
  gateway: {
    startAccount: async (ctx) => {
      const grammy = require("grammy");
      const bot = new grammy.Bot(ctx.account.config.botToken);
      
      bot.on("message", async (msg) => {
        // 处理消息
        const envelope = formatInboundEnvelope(msg);
        await ctx.gateway.broadcast(envelope);
      });
      
      await bot.start();
      
      // 返回清理函数
      return () => bot.stop();
    },
  },
  
  // Outbound 适配器
  outbound: {
    send: async (ctx, payload) => {
      const grammy = require("grammy");
      const bot = new grammy.Bot(ctx.account.config.botToken);
      
      await bot.api.sendMessage(ctx.target.chatId, payload.text);
      
      return { success: true };
    },
  },
});

export default telegramPlugin;
```

### 5.4 Provider 流式处理

```typescript
import type { StreamingSupport } from "openclaw/plugin-sdk";

const streamingConfig: StreamingSupport = {
  // 创建传输层
  createTransport: (params) => {
    return {
      async request(endpoint, body) {
        const response = await fetch(`${params.baseUrl}${endpoint}`, {
          method: "POST",
          headers: {
            "Content-Type": "application/json",
            Authorization: `Bearer ${params.apiKey}`,
          },
          body: JSON.stringify(body),
        });
        
        if (!response.ok) {
          throw new Error(`HTTP ${response.status}: ${response.statusText}`);
        }
        
        return response.body!;
      },
    };
  },
  
  // 解析流式响应
  parseStream: async (readable, callbacks) => {
    const reader = readable.getReader();
    const decoder = new TextDecoder();
    
    try {
      while (true) {
        const { done, value } = await reader.read();
        if (done) break;
        
        const text = decoder.decode(value);
        const lines = text.split("\n");
        
        for (const line of lines) {
          if (line.startsWith("data: ")) {
            const data = JSON.parse(line.slice(6));
            
            // 文本增量
            if (data.delta?.content) {
              await callbacks.onTextDelta?.(data.delta.content);
            }
            
            // 推理内容
            if (data.delta?.reasoning) {
              await callbacks.onReasoningDelta?.(data.delta.reasoning);
            }
            
            // Token 使用统计
            if (data.usage) {
              await callbacks.onUsage?.(data.usage);
            }
          }
        }
      }
    } finally {
      reader.releaseLock();
    }
  },
};
```

```typescript
// src/plugins/loader.ts

/**
 * 插件加载器
 * 
 * 负责扫描、验证和加载所有已安装的插件
 */
export class PluginLoader {
  private plugins = new Map<string, LoadedPlugin>();
  
  /**
   * 加载所有插件
   */
  async loadAllPlugins(config: OpenClawConfig): Promise<PluginRuntime> {
    // ========== 阶段 1: 扫描已安装插件 ==========
    const installedPlugins = await this.scanInstalledPlugins();
    log.info(`Found ${installedPlugins.length} installed plugins`);
    
    // ========== 阶段 2: 根据配置筛选启用的插件 ==========
    const enabledPlugins = this.filterEnabledPlugins(
      installedPlugins,
      config.plugins
    );
    log.info(`Enabled ${enabledPlugins.length} plugins`);
    
    // ========== 阶段 3: 加载 Provider 插件 ==========
    const providers = await this.loadProviderPlugins(enabledPlugins);
    
    // ========== 阶段 4: 加载 Channel 插件 ==========
    const channels = await this.loadChannelPlugins(enabledPlugins);
    
    // ========== 阶段 5: 注册全局 Hooks ==========
    await this.registerGlobalHooks(enabledPlugins);
    
    return {
      providers,
      channels,
      hooks: getGlobalHookRunner(),
    };
  }
  
  /**
   * 扫描已安装插件
   */
  private async scanInstalledPlugins(): Promise<DiscoveredPlugin[]> {
    const plugins: DiscoveredPlugin[] = [];
    
    // 扫描 extensions 目录
    const extensionsDir = path.join(process.cwd(), "extensions");
    const entries = await fs.promises.readdir(extensionsDir, {
      withFileTypes: true,
    });
    
    for (const entry of entries) {
      if (entry.isDirectory()) {
        const pluginPath = path.join(extensionsDir, entry.name);
        const manifest = await this.loadPluginManifest(pluginPath);
        
        if (manifest) {
          plugins.push({
            id: manifest.name,
            path: pluginPath,
            manifest,
          });
        }
      }
    }
    
    return plugins;
  }
  
  /**
   * 加载插件 Manifest
   */
  private async loadPluginManifest(pluginPath: string): Promise<PluginManifest | null> {
    try {
      const manifestPath = path.join(pluginPath, "package.json");
      const manifestContent = await fs.promises.readFile(manifestPath, "utf-8");
      const manifest = JSON.parse(manifestContent);
      
      // 验证必需字段
      if (!manifest.name || !manifest.main) {
        log.warn(`Invalid manifest in ${pluginPath}`);
        return null;
      }
      
      return manifest;
    } catch (error) {
      log.warn(`Failed to load manifest from ${pluginPath}:`, error);
      return null;
    }
  }
}
```

```typescript
interface SandboxOptions {
  enabled: boolean;
  timeoutMs: number;
  memoryLimitMb: number;
  allowedPaths: string[];
  blockedBins: string[];
}

const DEFAULT_SANDBOX_OPTIONS: SandboxOptions = {
  enabled: false,  // 默认不启用，除非明确配置
  timeoutMs: 300_000,  // 5 分钟
  memoryLimitMb: 512,
  allowedPaths: ["/tmp", "/workspace"],
  blockedBins: ["rm", "sudo", "mkfs", "dd"],
};

export async function runPluginInSandbox(
  pluginPath: string,
  options: SandboxOptions = DEFAULT_SANDBOX_OPTIONS
): Promise<SandboxResult> {
  // 1. 创建子进程
  const child = fork(path.join(__dirname, "sandbox-worker.js"), {
    execArgv: [`--max-old-space-size=${options.memoryLimitMb}`],
  });
  
  // 2. 设置超时
  const timeout = setTimeout(() => {
    child.kill("SIGKILL");
  }, options.timeoutMs);
  
  // 3. 监听结果
  return new Promise((resolve, reject) => {
    child.on("message", (result) => {
      clearTimeout(timeout);
      resolve(result);
      child.disconnect();
    });
    
    child.on("error", (error) => {
      clearTimeout(timeout);
      reject(error);
      child.kill("SIGKILL");
    });
    
    // 4. 发送加载指令
    child.send({ type: "load", pluginPath });
  });
}
```

```typescript
// src/plugins/rate-limit.ts

class PluginRateLimiter {
  private limits = new Map<string, RateLimitConfig>();
  private counters = new Map<string, Counter>();
  
  registerLimit(pluginId: string, config: RateLimitConfig) {
    this.limits.set(pluginId, config);
  }
  
  async checkLimit(pluginId: string, action: string): Promise<RateLimitResult> {
    const config = this.limits.get(pluginId);
    if (!config) {
      return { allowed: true };  // 无限制配置则允许
    }
    
    const key = `${pluginId}:${action}`;
    let counter = this.counters.get(key);
    
    if (!counter) {
      counter = { count: 0, resetAt: Date.now() + config.windowMs };
      this.counters.set(key, counter);
    }
    
    // 重置窗口
    if (Date.now() > counter.resetAt) {
      counter.count = 0;
      counter.resetAt = Date.now() + config.windowMs;
    }
    
    // 检查是否超限
    if (counter.count >= config.maxRequests) {
      return {
        allowed: false,
        retryAfter: counter.resetAt - Date.now(),
      };
    }
    
    // 增加计数
    counter.count++;
    return { allowed: true };
  }
}

interface RateLimitConfig {
  windowMs: number;      // 时间窗口 (毫秒)
  maxRequests: number;   // 最大请求数
}

interface RateLimitResult {
  allowed: boolean;
  retryAfter?: number;   // 重试等待时间 (毫秒)
}
```

```typescript
// extensions/domestic-ai/index.ts

import { defineSingleProviderPluginEntry } from "openclaw/plugin-sdk/provider-entry";

export default defineSingleProviderPluginEntry({
  id: "domestic-ai",

            Authorization: `Bearer ${ctx.apiKey}`,
          },
        });
        
        if (!response.ok) {

        }
        
        const data = await response.json();
        
        return {
          models: data.models.map((m: any) => ({

            contextWindow: m.max_context_length,
            pricing: {
              input: m.price_input,

            },
            capabilities: {
              chat: true,
              vision: m.supports_vision,
              tools: m.supports_tools,
              reasoning: m.supports_reasoning,
            },
          })),
        };
      },
    },
    
    // 自定义流式处理
    streaming: {
      createTransport: (params) => {
        return {
          async request(endpoint, body) {
            const response = await fetch(`https://api.domestic-ai.com/v1${endpoint}`, {
              method: "POST",
              headers: {
                "Content-Type": "application/json",
                Authorization: `Bearer ${params.apiKey}`,
              },
              body: JSON.stringify(body),
            });
            
            if (!response.ok) {
              throw new Error(`HTTP ${response.status}: ${response.statusText}`);
            }
            
            return response.body!;
          },
        };
      },
      
      parseStream: async (readable, callbacks) => {
        const reader = readable.getReader();
        const decoder = new TextDecoder();
        
        try {
          while (true) {
            const { done, value } = await reader.read();
            if (done) break;
            
            const text = decoder.decode(value);
            const lines = text.split("\n");
            
            for (const line of lines) {
              if (line.startsWith("data: ")) {
                const data = JSON.parse(line.slice(6));
                
                if (data.choices?.[0]?.delta?.content) {
                  await callbacks.onTextDelta?.(data.choices[0].delta.content);
                }
                
                if (data.usage) {
                  await callbacks.onUsage?.(data.usage);
                }
              }
            }
          }
        } finally {
          reader.releaseLock();
        }
      },
    },
  },
});
```

### 9.3 示例 3: 创建 Hook 插件

**场景**: 为所有 Agent 请求添加审计日志和性能监控

```typescript
// extensions/audit-hook/index.ts

import type { HookEntry, OpenClawPluginApi } from "openclaw/plugin-sdk";

const performanceHook: HookEntry = {
  name: "before_agent_start",
  handler: async (event, context) => {
    const startTime = Date.now();
    
    // 记录请求开始
    console.log("[AUDIT] Agent starting:", {
      runId: context.runId,
      agentId: context.agentId,
      sessionId: context.sessionId,
      promptLength: event.prompt.length,
      messageCount: event.messages?.length || 0,
      timestamp: new Date().toISOString(),
    });
    
    // 存储开始时间到上下文 (供 after_turn 使用)
    context.metadata = context.metadata || {};
    context.metadata.startTime = startTime;
  },
};

const afterTurnHook: HookEntry = {
  name: "after_turn" as any,  // 需要类型断言
  handler: async (event, context) => {
    const endTime = Date.now();
    const startTime = context.metadata?.startTime || endTime;
    const duration = endTime - startTime;
    
    // 记录性能指标
    console.log("[PERF] Agent turn completed:", {
      runId: context.runId,
      duration: `${duration}ms`,
      tokensUsed: event.tokenUsage,
      messagesAdded: event.messagesAdded,
      compactionPerformed: event.compactionPerformed,
    });
    
    // 发送到监控系统
    await sendToMonitoring({
      event: "agent_turn_complete",
      metrics: {
        duration,
        tokens: event.tokenUsage,
      },
      context,
    });
  },
};

async function sendToMonitoring(data: unknown) {
  // 实现发送到监控系统的逻辑
  if (process.env.MONITORING_ENDPOINT) {
    await fetch(process.env.MONITORING_ENDPOINT, {
      method: "POST",
      headers: { "Content-Type": "application/json" },
      body: JSON.stringify(data),
    });
  }
}

// 插件入口
export default {
  register(api: OpenClawPluginApi) {
    api.registerHook(performanceHook.name as any, performanceHook.handler);
    api.registerHook(afterTurnHook.name as any, afterTurnHook.handler);
  },
};
```
